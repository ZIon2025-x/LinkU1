1. **双击 `pitch-deck.html`** 用浏览器打开（Chrome / Edge / Safari 均可）
2. **按 `F11` 全屏**
3. 键盘操作：
   - `←` / `→` 切页
   - `Space` 下一页
   - `Home` / `End` 跳首尾
4. **右上角按钮**切换中文 / English
5. 移动端支持左右滑动翻页

**离线可用** —— 单 HTML 文件，无任何外部依赖，断网也能演示。

## 🎯 不同赛道的内容侧重建议 / Track-Specific Tips

### 互联网+（高教主赛道）
- 强调**社会价值**（海外华人弱势群体保护、跨文化连接）
- 突出**创新性**（服务商品化模式、AI 助手、五合一闭环）

### 互联网+（红旅赛道 · 青年红色筑梦之旅）
- 不太适配本项目（红旅赛道偏向农村/老区帮扶），如必须参赛可强调"海外华人乡愁连接"角度

### 挑战杯（创业计划竞赛）
- 加强**商业可行性**论证：单位经济、财务模型、退出路径
- 加强**竞争分析**与**护城河**论述

### 国际赛道
- 直接提交英文版 `BUSINESS_PLAN_EN.md`
- Pitch deck 默认切到 English

## 🛠️ 建议的迭代方向 / Suggested Iterations

如果时间允许：

1. **真实数据替换**：把保守预测换成实际后台数据（注册用户、订单数、GMV、留存率）

5. **Q&A 预演**：准备 20 个常见评委问题（市场规模算法、竞争壁垒、获客模型、团队稳定性、退出路径）
