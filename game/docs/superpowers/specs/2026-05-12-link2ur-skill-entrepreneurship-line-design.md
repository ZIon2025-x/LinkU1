## 2 · 双阶段叙事弧 (新增)

### 2.1 Phase 1 (W2-W22) · "懂 AI 的留学生学姐/学长"

玩家入读伦敦 MSc 第一年。本来是来读书的,但发现身边一堆同道有 AI 内容需求 + 自己手熟 ChatGPT/Midjourney/Sora。从 Welcome Week 开始用 Link2Ur 接小单。

**典型服务包**:
- AI 双语字幕 / 短视频本地化 (£50-100)
- 小红书/IG 内容矩阵 AI 拳 (£100-300)
- AI logo + IG 视觉品牌套装 (£200-500)
- 留学生申请视频 / Show Reel AI 加工 (£40-200)

**典型客户痛点**: 自己不会用 AI / 没时间学 / 双语本地化是 bottleneck。

### 2.2 Phase 1 → Phase 2 转型 hook (Ch 3-4 期间)

**双引擎触发**:

1. **客户成长引擎** (W17-W20): 某个 Phase 1 网红客户被中资品牌签约 → 品牌方通过她找到玩家做英国市场素材 → 玩家**第一次接触品牌级单**(£800+,超过日常上限)
2. **Y 姐推动引擎** (W21-W22): Y 姐看到玩家给留学生做的好作品 → Sketch 邀请的 framing 变成 "你做的留学生级别 portfolio 我看了。技术上可以做品牌级,但你一个人接不过来。来聊聊。"

两引擎同时拉玩家 pivot,让转型不是"系统强制",而是真实压力导致。

### 2.3 Phase 2 (W23-W52) · "留学生背景的跨境广告人"

转型后玩家服务对象升级 + 单价升级 + 客户群体重定义。

**典型服务包**:
- 中资 DTC 品牌进 UK · 双语本地化 + Meta/Google ads campaign (£1000-3000)
- 英资 SMB 进华 · 小红书/微信运营 + 双语本地化 (£500-2000)
- 中国背景 VC-backed startup 出海首批英文素材 + GTM (£800-2500)
- 高端创意代理外包 · AI 子项目 (£400-1500)

**Phase 1 客户大部分流失** (网红客单太低,玩家精力分配不过来),但少数(2-3 个)成长跟随玩家到 Phase 2 - 形成"老客户陪你一起成长"的叙事高潮。

## 4 · 4 个核心机制 specs

### 4.1 机制 A · 回头客系统 (`link2urRepeat.js`)

**触发**: 完成任务后调用 `maybePromoteToRepeat(state, completedTask)`

**4 层关系递进阈值** (与 Phase 无关,通用):

| 阈值 | 状态 | 解锁 |
|---|---|---|
| 1 单 + 评分≥4.5 | first_impression | "客户记住你了" 提示 |
| 2 单 + 评分≥4.8 | repeat_unlocked | 解锁该 customer inbox 任务 |
| 4 单 + 评分≥4.8 | fan_unlocked | 解锁专属对话场景 |
| 6 单 + 评分≥4.9 | loyal | 触发独家事件(签约/品牌升级/感谢信) |

**Phase-aware 匹配**: customerAffinityType 不仅匹配 task type,还匹配 Phase:
- Phase 1 客户在 Phase 1 期间积极晋升
- Phase 2 客户只在 Phase 2 开启后才上线
- 跨阶段客户 (2-3 个) 在 Phase 1 晋升,Phase 2 继续 work

const SHOULD_TRIGGER_Y_INVITATION = (state) => {
  return state.link2urClashCount >= 3
      && Math.ceil(state.day / 7) >= 21
      && state.link2urRating >= 4.7
