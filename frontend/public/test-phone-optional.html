手机号可选功能测试

测试注册时手机号字段是否为可选。

1. 不提供手机号

{ "name": "testuser_no_phone", "email": "test_no_phone@example.com", "password": "12345678a" }

2. 提供空字符串手机号

{ "name": "testuser_empty_phone", "email": "test_empty_phone@example.com", "password": "12345678a", "phone": "" }

3. 提供null手机号

{ "name": "testuser_null_phone", "email": "test_null_phone@example.com", "password": "12345678a", "phone": null }

4. 提供有效手机号

{ "name": "testuser_valid_phone", "email": "test_valid_phone@example.com", "password": "12345678a", "phone": "1234567890" }
