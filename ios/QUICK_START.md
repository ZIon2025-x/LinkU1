# iOS 应用快速开始指南 - 用户端

> **注意**: 这是 LinkU 的用户端 iOS 应用，不包含客服和管理员功能。

## 🚀 快速测试步骤

### 1. 在 Xcode 中打开项目

按照 `SETUP.md` 中的步骤创建项目并添加文件。

### 2. 配置 API 地址

**重要**: 在运行前必须配置正确的 API 地址！

打开 `LinkU/Utils/Constants.swift`，确认已配置：

```swift
static let apiBaseURL = "https://api.link2ur.com"
static let wsBaseURL = "wss://api.link2ur.com"
```

API 和 WebSocket 地址已配置完成。

### 3. 配置权限

确保 `Info.plist` 中包含所有必要的权限描述（已在代码中配置）。

### 4. 运行项目

1. 选择目标设备（模拟器或真机）
2. 按 `⌘R` 运行
3. 首次运行会显示登录界面

### 5. 测试功能

#### 登录测试
- 使用现有的用户账号登录
- 登录成功后会自动连接 WebSocket

#### 功能测试清单（用户端功能）
- ✅ 登录/注册
- ✅ 任务列表浏览
- ✅ 任务详情查看
- ✅ 跳蚤市场浏览
- ✅ 消息收发（需要 WebSocket 连接）
- ✅ 个人中心（我的任务、我的发布、钱包）
- ✅ 设置与关于

**不包含的功能**:

- ❌ 管理员功能

## 🐛 常见问题

### WebSocket 连接失败
- 检查 API 地址是否正确
- 检查网络连接
- 查看 Xcode 控制台日志

### API 请求失败
- 检查 `baseURL` 配置
- 检查网络权限
- 查看错误日志

### 编译错误
- 确保所有文件都已添加到项目
- 检查文件是否在正确的 target 中
- 清理构建缓存：`Product > Clean Build Folder`
