# Link²Ur 应用名称指南

## 应用名称

**Link²Ur**（注意：² 是上标2，不是普通的数字2）

## 如何输入 ²（上标2）

### 方法 1：复制粘贴（最简单）

直接复制这个字符：**²**

### 方法 2：在 Mac 上输入

1. **使用特殊字符面板**：
   - 按住 `Option` 键，然后按 `2` 键
   - 或者：`Control + Command + Space` 打开字符查看器，搜索"上标2"

2. **使用 Unicode**：
   - Unicode 编码：`U+00B2`
   - 十六进制：`00B2`

### 方法 3：在 Windows 上输入

1. **使用 Alt 码**：
   - 按住 `Alt` 键
   - 输入 `0178`（数字键盘）
   - 松开 `Alt` 键

2. **使用字符映射表**：
   - 打开"字符映射表"（charmap.exe）
   - 搜索"上标2"或"superscript two"
   - 复制字符

### 方法 4：在 iOS/Android 上输入

1. **iOS**：
   - 长按数字 `2`，会显示上标选项
   - 或者使用第三方输入法

2. **Android**：
   - 使用支持特殊字符的输入法
   - 或复制粘贴

### 方法 5：在代码中使用

**HTML**：
```html
Link&sup2;Ur
或
Link&#178;Ur
或
Link&#xB2;Ur
```

**Unicode 转义**：
```
Link\u00B2Ur
```

**直接使用字符**：
```
Link²Ur
```

## 常见问题

### Q: ² 和普通的 2 有什么区别？

**A**: 
- **²** 是上标2（superscript 2），通常用于表示平方（如 m² = 平方米）
- **2** 是普通数字2
- 在应用名称中，² 表示 "to" 的谐音，同时保持品牌独特性

### Q: 用户如何搜索这个应用？

**A**: 
- 在 App Store 搜索关键词中必须包含 **"link2ur"**（普通数字2）
- 用户搜索 "link2ur" 或 "link to you" 都能找到应用
- 详见：`ios/APP_STORE_SEARCH_OPTIMIZATION.md`

### Q: 在代码中如何处理这个字符？

**A**: 
- 直接使用 Unicode 字符：`Link²Ur`
- 或使用 HTML 实体：`Link&sup2;Ur`
- 确保文件编码为 UTF-8

### Q: 在 URL 或文件名中如何使用？

**A**: 
- URL 中通常使用普通字符：`link2ur` 或 `link-to-you`
- 文件名中可以使用：`Link2Ur` 或 `Link-to-Ur`
- 避免在 URL 中使用特殊字符（可能导致编码问题）
