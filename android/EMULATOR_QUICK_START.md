# 模拟器快速开始（5分钟）

## ⚡ 超快速步骤

### 1. 打开设备管理器（10秒）
```
Tools > Device Manager
```

### 2. 创建模拟器（2分钟）
1. 点击 `Create Device`
2. 选择 `Pixel 5`
3. 选择 `API 33`（如果没有，点击 Download 下载）
4. 点击 `Next` > `Finish`

### 3. 启动模拟器（1分钟）
1. 在列表中点击播放按钮（▶️）
2. 等待启动完成

### 4. 运行应用（1分钟）
1. 在 Android Studio 顶部选择模拟器
2. 点击运行按钮（▶️）
3. 等待应用安装并启动

## ✅ 完成！

应用现在应该在模拟器中运行了。

## 🐛 如果遇到问题

- **模拟器启动慢**: 这是正常的，首次启动需要几分钟
- **应用无法运行**: 检查 Logcat 中的错误信息
- **网络连接失败**: 确保模拟器可以打开浏览器访问网页
